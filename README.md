# Apple Clock

一个极简风格的数字时钟网页，具有 Apple 审美设计。

## 功能特性

- **实时时钟** - 显示时:分:秒，SF Pro 字体
- **名人名言** - 每分钟自动更换，淡入淡出效果
- **2.5D 视差** - 鼠标移动时产生立体效果
- **Siri 波浪** - 底部装饰性正弦波浪动画
- **深色模式** - 自动检测系统主题，支持手动切换
- **个性设置** - 可自定义字体大小、位置、视差强度

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 点击右上角按钮切换深色/浅色模式
3. 点击右下角齿轮图标打开设置面板
4. 在设置中可以：
   - 调整时钟和名言的字体大小
   - 调整内容垂直位置
   - 调节视差强度
   - 开启/关闭波浪效果

## 发布到 Wallpaper Engine

1. 打开 Wallpaper Engine 编辑器
2. 将 `index.html` 拖入编辑器
3. 设置背景色为透明
4. 保存并发布

## 技术栈

- 纯 HTML/CSS/JavaScript
- CSS Custom Properties 主题系统
- Canvas 动画
- requestAnimationFrame 流畅动画
